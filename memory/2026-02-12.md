## Telegram Streaming Issue (2026-02-12)
- `streamMode: "partial"` enables Telegram draft streaming (can cause fragmented messages)
- `streamMode: "off"` disables draft streaming
- `streamMode: "block"` uses chunked draft updates
- Draft streaming is separate from block streaming
